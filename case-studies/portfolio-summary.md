## Positioning

**Present as:** Automation Engineer · Backend Systems Engineer · Workflow Architect · API Integration Specialist · AI Automation Specialist

**Lead with:** Engineering patterns and system design—not industry domain

**Avoid presenting as:** No-code builder · Citizen developer · Workflow operator · Industry-specific consultant
