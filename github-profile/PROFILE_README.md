## About

I design, integrate, and scale **production workflow systems** — writing custom JavaScript, API contracts, validation logic, and orchestration patterns for high-volume operational teams.

Automation platforms (**n8n**, **Workato**) are execution layers. The engineering is in the **architecture**, **integrations**, and **code**.

```text
Slack / Gmail / Webhooks  →  Intent Router  →  Sub-workflows  →  REST APIs  →  Structured Response
                                    ↑
                              Claude / OpenAI (structured outputs)
```

**Currently:** Available for remote Automation · Integration · Backend engineering roles.
