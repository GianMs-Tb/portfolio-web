# Intelligent Follow-Up Engine

**Rules-driven stakeholder outreach with anti-spam guardrails**

## Overview

Policy-aware outreach engine mapping chat identities to platform users, selecting templates by process tier and sequence, enforcing 48-hour cooldowns.

## Key Metrics

- 5–8 min saved per outreach
- 48h cooldown enforced
- 4 dynamic template routes
- 25+ operators served

## Reusable Patterns

- Cross-system identity mapping (chat → CRM)
- JavaScript rules engine for template selection
- Communication history analysis with per-sender cooldown
- Skip/block guardrails with structured responses
- Sub-workflow I/O contract for orchestrator integration
